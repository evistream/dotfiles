dotfiles for CentOS

# how to install
```
sudo yum install -y git
cd ~
git clone https://github.com/evistream/dotfiles
vim ./dotfilese/initialize_CentOS7.sh
su
# sudo -u root sh ./dotfiles/initialize_CentOS7.sh
sh ./dotfiles/initialize_CentOS7.sh
```
